# ⛓️ Crypto Academy

A plain-language course on blockchain, wallets and on-chain safety. Free, no sign-up, progress saved in your browser.

**Live demo:** https://skrizidev.github.io/crypto-academy/

## Pages
- **Home** — landing with course overview
- **Lessons** — 10 short lessons with click-to-complete progress tracking (`localStorage`)
- **Glossary** — searchable crypto terms

## Tech
Vanilla HTML / CSS / JavaScript. Educational content only — not financial advice.
